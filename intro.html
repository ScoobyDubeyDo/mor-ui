MOR UI

CODE LESS, STYLE MORE.

MOR is an open-source styling framework that helps you develop a fast and easy user interface. Spend less time in designing and focus more on logic without putting much effort into UI.

Customizing Themes

Although MOR aims to satisfy the requirements of its users as much as possible, but sometimes the predefined styles are not enough.

To overload the MOR's base classes, users can add their own classes to existing classes to style them separately.
