Rating

Rating components can be used as read-only badge or in reviews section. Can be used in reviews section as form too.

Imports

Rating as a input

The initial value of the rating component is 0 but once the user starts using it the value cannot go below 1.

You can use as many stars as you want but there must be at least 2 stars for this to work.

This Component requires javaScript to function, Add this script to your html document.
