Alert

An Alert displays a short, important message in a way that draws the attention of the user without interrupting the task of the user. Our alerts are available for any length of text, along with an optional closure button.

Basic alerts

The alert offers four severity levels that set a distinctive icon and color.

Alert with actions

An alert may have an action, such as a close or undo button. It is displayed following the message, at the end of the alert.

Add alert-button class to the action element. Action buttons are optional to have.
